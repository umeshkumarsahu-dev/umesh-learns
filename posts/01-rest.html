📌 What is REST?

REST stands for Representational State Transfer. It is a style of architecture for designing networked applications using HTTP.

Common REST Methods:

  • GET → Retrieve Data
  • POST → Send Data
  • PUT → Update Data
  • DELETE → Delete Data

🌐 Example:

GET     /users          → get all users
POST    /users          → create new user
PUT     /users/1        → update user 1
DELETE  /users/1        → delete user 1

✅ REST Principles:

  • Stateless
  • Client-Server Separation
  • Cacheable
  • Uniform Interface
