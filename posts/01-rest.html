🧠 What is REST API?

Date: 22 June 2025

📌 REST Meaning

REST stands for Representational State Transfer. It is an architecture style for designing networked applications.

💡 REST API (Application Programming Interface)

REST APIs use HTTP methods to perform actions on resources, like:

  • GET – Retrieve data
  • POST – Create new data
  • PUT – Update existing data
  • DELETE – Remove data

🔗 Example:

For a REST API for "users", these could be the routes:

  • GET `/users` → List all users
  • POST `/users` → Add new user
  • PUT `/users/1` → Update user with ID 1
  • DELETE `/users/1` → Delete user with ID 1

✅ Key Characteristics of REST APIs:

  • Stateless
  • Uses standard HTTP methods
  • Resource-based (URI)
  • Supports multiple formats (usually JSON)

🧠 Summary

REST is simple, scalable, and widely used in modern web applications and backend systems.
